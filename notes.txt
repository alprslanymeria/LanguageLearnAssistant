WITH-ASP-NET-CORE-CLEAN-ARCH
_________________________________

- Prisma olmayacak, database backend tarafından yürütülecek.
- Action metotlar olmayacak. Action metotlar backend tarafında taşınacak.
- Dockerfile burada kalacak. Nextjs image'ı docker hub'a yüklenecek. docker compose dosyası ve otel collector
config dosyası, prometheus yaml dosyası backend tarafından tanımlanacak.
- proxy.js dosyası kontrol edilir.
- zod schema kaldırılır. Backend tarafında fluent validation ile bunların kontrolü sağlanır.


- Backend tarafında controller'lar yazılacak, ilgili entity'ler için servisler ve repository'ler yazılacak.
- request ve response modelle yazılır.
